# Product Spec: Social Media Strategy

**Status**: Active
**Owner**: ContentAgent
**Last Updated**: 2026-02-18

## Overview

OpenClaw maintains a multi-platform presence targeting aerospace engineers, satellite operators, space enthusiasts, and crypto communities. Content is generated by the ContentAgent using self-improving skill files.

## Platform Matrix

| Platform | Audience | Content Type | Cadence |
| --- | --- | --- | --- |
| X/Twitter | Space + Crypto communities | Facts, visualizations, alerts, memes | 3-5 posts/day |
| TikTok | Gen Z + space enthusiasts | Photo slideshow carousels | 1-3 posts/day |
| LinkedIn | Aerospace professionals | Industry insights, case studies | 2-3 posts/week |
| YouTube | Learners + engineers | Tutorials, explainers, walkthroughs | 1-2 videos/week |
| Discord | Community hub | Real-time support, Q&A, AMAs | Always-on |
| Threads | Casual engagement | Space commentary, behind-the-scenes | 1-2 posts/day |
| Farcaster | Web3-native community | Token launches, DeFi, governance | 2-3 posts/week |

## Content Pillars

1. **Orbital Mechanics Education** (40%) — Making astrodynamics accessible
2. **Space Debris Awareness** (20%) — The crisis and how OpenClaw helps
3. **Product Updates** (15%) — Feature launches, demos, tutorials
4. **Community** (15%) — Token milestones, governance, events
5. **Industry News** (10%) — Reactions to launches, debris events, policy

## Bot Persona

- **Name**: OpenClaw (the bot IS the brand)
- **Voice**: Expert but approachable
- **Personality**: PhD student who loves explaining orbital mechanics to anyone
- **Tone**: Educational, enthusiastic, occasional space puns
- **Values**: Open-source, space sustainability, democratization of knowledge
- **Never**: Hard-sell, spam, engage in arguments, share unverified claims

## Content-to-Conversion Funnel

```
Awareness (social media impressions)
  → Interest (click-through to site/demo)
    → Trial (free tier signup, connect wallet)
      → Conversion (buy tokens OR subscribe)
        → Retention (ongoing usage, community participation)
```

Track at each stage. ContentAgent diagnoses where the funnel is leaking using the 2x2 matrix (see content-generation.md).

## Tools

- **Image generation**: OpenAI gpt-image-1.5 ($0.50/slideshow, $0.25 batch)
- **TikTok posting**: Postiz API (drafts, then manual audio + publish)
- **Scheduling**: Postiz + agent-driven scheduling
- **Analytics**: TikTok via Postiz, X native analytics, YouTube Studio
